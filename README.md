### Обраний варіант
Менеджер завдань для студентів.

### Головна дія
Створення нового завдання.

### Щасливий шлях
Користувач відкриває головну сторінку застосунку.
Натискає кнопку «Додати завдання».
Вводить назву завдання.
Зберігає його.
Завдання з’являється у списку активних справ.

### Альтернативні стани
 Порожній список - відображається повідомлення «У вас ще немає завдань».
 Порожнє поле вводу - система не дозволяє зберегти та підсвічує поле.

### Структура сторінки (вайрфрейм)
 Шапка з назвою застосунку
 Центральний блок із коротким описом
 Головна кнопка «Додати завдання»
 Список завдань
 Допоміжна зона

### Правила сітки
Використовується єдина колонкова структура.
Базовий відступ між блоками — 16px.
Заголовки, кнопки та список вирівняні по одній лівій осі.

### Кольорова палітра:
- світло-рожеві відтінки для основних елементів інтерфейсу
- персикові та пудрові кольори для акцентів
- приглушені сірі тони для фону та другорядних елементів
- нейтральні кольори для зменшення візуального навантаження
### Типографія:
- сучасні sans-serif шрифти
- висока читабельність тексту
- чітка ієрархія заголовків і основного тексту
- адаптивність для desktop та mobile версій
